# Community Radio Contributor Hub

An open-source starter framework that helps community radio stations guide volunteers, remote contributors, syndicated program providers, and community organizations through the right level of orientation, station policy, and content submission expectations.

## Why this exists

Community radio stations often receive interest from people with very different roles:

- Local volunteers who may need a full learning pathway
- Remote contributors who submit prerecorded content
- Syndicated program providers who need station-specific policies
- Community organizations that want PSAs, interviews, or announcements
- Technical producers who support studio or live workflows

This project is designed to reduce irrelevant or off-mission content submissions while making it easier for contributors to understand what the station needs before they contact staff or submit content.

## MVP scope

This first version is intentionally static and lightweight.

Included pages:

- Home / contributor path selection
- Station policies
- Content submission guide
- Volunteer learning pathway
- Production exercises
- Adaptation guide

## What this is not

This is not a podcast launch course.
This is not an employee training platform.
This is not a full learning management system.

It is a reusable community radio contributor guide that can later be extended with quizzes, completion tracking, or a database-backed submission workflow.

## Local use

Open `index.html` in a browser.

No build tools are required for the MVP.
